## Blink
Blink is the best real-time communications client using SIP protocol. You can use it with many SIP providers, on the LAN using Bonjour and SIP2SIP free service.

#### Features
* Voice and Video	
* IM and Presence	
* File Transfers	
* Screen Sharing	
* Conferencing
