## Super Mario Bros Java (Portable)

![Screenshot of Super Mario Bros Java](https://a.fsdn.com/con/app/proj/super-mario-bros-java/screenshots/1(1).png)	

###Buttons:
* Z = Fire / Speed
* X = Jump
* Arrow Left = Go Left
* Arrow Right = Go Right
* Arrow Down = Crouch / Go inside Pump(Downwards)
* T = Take ScreenShot
* I = Info
